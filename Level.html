(static) addParticle(particleType, x, y, z, velX, velY, velZ, size)

Add a particle in the world in the desired position. Based on ParticleType.java from BlockLauncher, the available particle types are: blockcrack 1; crit 2; smoke 3; cloud 4; flame 5; lava 6; redstone 8; itemcrack 9; snowballpoof 10; mobFlame 13; heart 14; Particle types can also be accessed by writing ParticleType.nameOfTheParticle, for example "ParticleType.redstone".
Parameters:
Name Type Description
particleType int The particle type to add
x int X coordinate
y int Y coordinate
z int Z coordinate
velX float The speed of the particle on the X axis
velY float The speed of the particle on the Y axis
velZ float The speed of the particle on the Z axis
size int The size of the particle (not all particles support size)
